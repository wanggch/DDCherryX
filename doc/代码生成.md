
**代码预览与下载**

- 在线预览生成的代码
- 下载 ZIP 压缩包
- 生成到自定义路径

### 技术特点：

- 使用 **Velocity** 模板引擎根据模板文件渲染后端代码
- 支持 **动态数据源** 切换
- 集成 **Anyline** 实现多数据库元数据查询
- 自动生成符合项目规范的标准化代码
- 可以在yml配置文件中配置作者、包路径、是否去掉表前缀、表前缀。

### 核心功能：

1. **数据库表管理**
   - 连接多数据源，查询数据库表结构
   - 导入表结构到代码生成配置
   - 同步数据库表结构变更
2. **代码生成**
   - 生成内容包括：
     - **后端代码**：Entity、VO、BO、Mapper、Service、Controller
     - **配置文件**：MyBatis XML、SQL 菜单脚本
3. **代码预览与下载**
   - 在线预览生成的代码
   - 下载 ZIP 压缩包
   - 生成到自定义路径
4. **多数据库支持**
   - 使用 Anyline 框架支持 100+ 种数据库
   - 默认支持：MySQL、Oracle、PostgreSQL、SQL Server
5. **智能字段处理**
   - 自动识别字段类型（字符串、数字、日期等）
   - 自动推断表单控件类型（输入框、下拉框、日期选择器等）
